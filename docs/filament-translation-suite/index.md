# Filament Translation Suite

The only translation tool your Filament app needs.

Every Laravel app has two translation worlds — UI strings in lang/ files, and content in translatable models. Other tools make you choose one. We handle both, natively inside your Filament panel.

## Key Features

- 🌍 **File & UI Translations** — Import existing lang/ files for safe database editing. Auto-scan your codebase for missing translation keys.
- 🧠 **Content Model Localization** — Auto-discovers every Spatie translatable model. Bulk translate entire tables via background queues.
- ⚡ **Zero-Config Form Morphing** — Every translatable field automatically gets locale tabs. No manual setup.
- 🤖 **Machine Translation Engine** — DeepL, Google Cloud, ChatGPT, and Claude. Single field or bulk.
- 🌐 **Built for Teams** — Permission-gated translation portal, CSV round-trip, webhooks for Phrase/Crowdin/Lokalise.
- 📊 **Health Dashboard** — Unified coverage reporting with real-time activity feed.
- 🛡️ **Non-Destructive by Design** — Publishing merges rather than replaces. Automatic backups.
